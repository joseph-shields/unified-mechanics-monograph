# Unified Mechanics

Three PDFs.

**A World of Distinctions.** The monograph. How a sentence, asked to describe itself, becomes a universe. Read this one first, start to finish.

**Universopedia, The Observational Standard.** The companion reference: what a record is, what can honestly be said about a system, and where established science ends and the framework's added layer begins.

**Systems of Interpretation.** A short, honest record of throwing the framework at real systems to see where it grips and where it slides off. Exploration, not a verdict.

Every claim carries a grade: proved, conditional, proposed, or postulate. They are never mixed.

Joseph Shields, 2026.
